Q-22True / False
In Swift, a switch statement requires a break statement at the end of each case to prevent fall-through to the next case.
Swift's switch is different from C and Java. Each case automatically ends — there is no fall-through unless you explicitly use fallthrough.
Q-23Predict Output
What does this loop print? (List all values, one per line)
range.swift
for i in 1..<4 {
    print(i)
}
The half-open range 1..<4 goes from 1 up to but NOT including 4. The closed range 1...4 would include 4.
