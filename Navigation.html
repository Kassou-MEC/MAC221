🎯 Introduction to SwiftUI Animations

What are SwiftUI Animations?

SwiftUI Animations are powerful tools that allow you to create smooth, engaging motion in your iOS applications. They help guide user attention, provide feedback, and make your app feel more polished and professional.

Why Use Animations?

  • Enhanced User Experience: Animations make your app feel more responsive and intuitive
  • Visual Feedback: Provide clear feedback for user actions
  • Focus Attention: Guide users to important content or actions
  • Professional Polish: Well-executed animations make your app stand out

Core Animation Concepts

Implicit Animations

Automatically animate changes when state variables change. Use .animation() modifier.

Explicit Animations

Manually trigger animations with withAnimation blocks for precise control.

Transitions

Animate views as they appear or disappear from the screen.

Spring Animations

Create natural, physics-based animations with bounce and elasticity.

Note: SwiftUI uses a declarative syntax for animations, making them easier to implement and reason about compared to UIKit's imperative approach.

🔰 Basic Animations

Example 1: Simple Scale Animation

This basic example demonstrates how to animate the scale of a view when a state variable changes.

ScaleAnimation.swift
import SwiftUI

struct ScaleAnimation: View {
    @State private var isScaled = false
    
    var body: some View {
        VStack(spacing: 30) {
            Circle()
                .fill(Color.blue)
                .frame(width: 100, height: 100)
                .scaleEffect(isScaled ? 1.5 : 1.0)
                .animation(.easeInOut(duration: 0.5), value: isScaled)
            
            Button("Animate Scale") {
                isScaled.toggle()
            }
            .buttonStyle(.borderedProminent)
        }
        .padding()
    }
}
Scale

Explanation:

  • @State: Tracks the animation state
  • scaleEffect: Modifies the view's scale based on state
  • .animation(): Applies the animation to scale changes
  • .easeInOut: Animation timing curve (starts slow, speeds up, ends slow)
Example 2: Rotation and Color Animation

Animating multiple properties simultaneously with different timing curves.

MultiPropertyAnimation.swift
import SwiftUI

struct MultiPropertyAnimation: View {
    @State private var isRotated = false
    @State private var colorChanged = false
    
    var body: some View {
        VStack(spacing: 30) {
            RoundedRectangle(cornerRadius: 20)
                .fill(colorChanged ? Color.purple : Color.orange)
                .frame(width: 150, height: 150)
                .rotationEffect(.degrees(isRotated ? 180 : 0))
                .animation(
                    .spring(response: 0.6, dampingFraction: 0.8),
                    value: isRotated
                )
                .animation(
                    .easeInOut(duration: 1.0),
                    value: colorChanged
                )
            
            Button("Animate All") {
                isRotated.toggle()
                colorChanged.toggle()
            }
            .buttonStyle(.borderedProminent)
        }
        .padding()
    }
}
Multi

Explanation:

  • rotationEffect: Animates rotation using degrees
  • Multiple animations: Different animations for different properties
  • Spring animation: Creates bouncy, natural motion
  • Simultaneous changes: Multiple state changes in one action
Example 3: Implicit vs Explicit Animations

Understanding the difference between implicit and explicit animation approaches.

AnimationTypes.swift
import SwiftUI

struct AnimationTypes: View {
    @State private var implicitOffset: CGFloat = 0
    @State private var explicitOffset: CGFloat = 0
    
    var body: some View {
        VStack(spacing: 40) {
            // Implicit Animation
            VStack {
                Text("Implicit Animation")
                    .font(.headline)
                
                Circle()
                    .fill(Color.green)
                    .frame(width: 80, height: 80)
                    .offset(x: implicitOffset)
                    .animation(.easeInOut(duration: 1.0), value: implicitOffset)
                
                Button("Move") {
                    implicitOffset = implicitOffset == 0 ? 100 : 0
                }
            }
            
            // Explicit Animation
            VStack {
                Text("Explicit Animation")
                    .font(.headline)
                
                Circle()
                    .fill(Color.blue)
                    .frame(width: 80, height: 80)
                    .offset(x: explicitOffset)
                
                Button("Move") {
                    withAnimation(.spring(response: 0.6, dampingFraction: 0.7)) {
                        explicitOffset = explicitOffset == 0 ? 100 : 0
                    }
                }
            }
        }
        .padding()
    }
}
Implicit
Explicit

Explanation:

  • Implicit Animation: Automatically animates when state changes
  • Explicit Animation: Manually triggered with withAnimation
  • Control: Explicit animations give you more control over timing and scope
  • Use Cases: Implicit for simple changes, explicit for complex sequences

🔄 Transitions

Example 1: Basic View Transitions

Animating views as they appear and disappear from the screen.

BasicTransitions.swift
import SwiftUI

struct BasicTransitions: View {
    @State private var showSquare = false
    
    var body: some View {
        VStack(spacing: 30) {
            if showSquare {
                RoundedRectangle(cornerRadius: 15)
                    .fill(Color.purple)
                    .frame(width: 150, height: 150)
                    .transition(.scale.combined(with: .opacity))
            }
            
            Button(showSquare ? "Hide Square" : "Show Square") {
                withAnimation(.easeInOut(duration: 0.8)) {
                    showSquare.toggle()
                }
            }
            .buttonStyle(.borderedProminent)
        }
        .padding()
    }
}
Hello!

Explanation:

  • .transition(): Defines how a view appears/disappears
  • Built-in transitions: .scale, .slide, .opacity, .move(edge:)
  • Combined transitions: Use .combined(with:) to mix effects
  • withAnimation: Required to animate transitions

Example 3: Custom Transitions

Creating your own custom transitions for unique effects.

CustomTransitions.swift
import SwiftUI

extension AnyTransition {
    static var rotateAndFade: AnyTransition {
        .modifier(
            active: RotateFadeModifier(rotation: 90, opacity: 0),
            identity: RotateFadeModifier(rotation: 0, opacity: 1)
        )
    }
    
    static var bounce: AnyTransition {
        .modifier(
            active: BounceModifier(scale: 0.1),
            identity: BounceModifier(scale: 1)
        )
    }
}

struct RotateFadeModifier: ViewModifier {
    let rotation: Double
    let opacity: Double
    
    func body(content: Content) -> some View {
        content
            .rotationEffect(.degrees(rotation))
            .opacity(opacity)
    }
}

struct BounceModifier: ViewModifier {
    let scale: CGFloat
    
    func body(content: Content) -> some View {
        content
            .scaleEffect(scale)
    }
}

struct CustomTransitions: View {
    @State private var showCustomView = false
    
    var body: some View {
        VStack(spacing: 30) {
            if showCustomView {
                RoundedRectangle(cornerRadius: 20)
                    .fill(LinearGradient(
                        gradient: Gradient(colors: [.red, .orange]),
                        startPoint: .topLeading,
                        endPoint: .bottomTrailing
                    ))
                    .frame(width: 200, height: 200)
                    .transition(.rotateAndFade)
            }
            
            Button(showCustomView ? "Hide" : "Show Custom") {
                withAnimation(.easeInOut(duration: 1.0)) {
                    showCustomView.toggle()
                }
            }
            .buttonStyle(.borderedProminent)
        }
        .padding()
    }
}
Custom!

Explanation:

  • AnyTransition extension: Create reusable custom transitions
  • ViewModifier: Define the active and identity states
  • Active state: The view's appearance during transition
  • Identity state: The view's normal appearance

🚀 Advanced Animations

Example 1: Animation Matched Geometry Effect

Creating seamless animations between different views using matchedGeometryEffect.

MatchedGeometry.swift
import SwiftUI

struct MatchedGeometry: View {
    @State private var isExpanded = false
    @Namespace private var animation
    
    var body: some View {
        VStack {
            if isExpanded {
                // Expanded view
                RoundedRectangle(cornerRadius: 25)
                    .fill(Color.blue)
                    .frame(width: 300, height: 400)
                    .overlay(
                        Text("Detailed View")
                            .font(.title)
                            .foregroundColor(.white)
                    )
                    .matchedGeometryEffect(id: "shape", in: animation)
                    .onTapGesture {
                        withAnimation(.spring()) {
                            isExpanded.toggle()
                        }
                    }
            } else {
                // Collapsed view
                RoundedRectangle(cornerRadius: 10)
                    .fill(Color.blue)
                    .frame(width: 100, height: 100)
                    .overlay(
                        Text("Tap me")
                            .foregroundColor(.white)
                    )
                    .matchedGeometryEffect(id: "shape", in: animation)
                    .onTapGesture {
                        withAnimation(.spring()) {
                            isExpanded.toggle()
                        }
                    }
            }
        }
        .padding()
    }
}
Tap me

Explanation:

  • @Namespace: Creates an animation namespace
  • matchedGeometryEffect: Links two views for seamless animation
  • Same ID: Both views share the same identifier
  • Smooth transitions: Creates fluid morphing between views
Example 2: Animation TimelineView

Creating complex animations that change over time using TimelineView.

TimelineAnimation.swift
import SwiftUI

struct TimelineAnimation: View {
    var body: some View {
        TimelineView(.animation) { timeline in
            Canvas { context, size in
                let time = timeline.date.timeIntervalSinceReferenceDate
                let angle = time.remainder(dividingBy: 2)
                
                // Draw multiple rotating elements
                for i in 0..<5 {
                    let rotation = Angle(radians: angle + Double(i) * .pi / 2.5)
                    let x = size.width / 2 + cos(rotation.radians) * 100
                    let y = size.height / 2 + sin(rotation.radians) * 100
                    
                    var elementContext = context
                    elementContext.translateBy(x: x, y: y)
                    elementContext.rotate(by: rotation)
                    
                    let rect = CGRect(x: -15, y: -15, width: 30, height: 30)
                    elementContext.fill(
                        Path(roundedRect: rect, cornerRadius: 6),
                        with: .color(Color(hue: Double(i) / 5, saturation: 0.8, brightness: 0.9))
                    )
                }
            }
        }
        .frame(height: 300)
    }
}

Explanation:

  • TimelineView: Updates content based on a schedule
  • .animation: Schedule that updates frequently for smooth animation
  • Canvas: Low-level drawing for complex graphics
  • Mathematical animations: Use trigonometry for circular motion
Example 3: Phase Animations

Creating multi-step animations using phase animator.

PhaseAnimator.swift
import SwiftUI

struct PhaseAnimator: View {
    var body: some View {
        VStack(spacing: 30) {
            // Simple phase animation
            PhaseAnimator([0, 1, 2]) { phase in
                RoundedRectangle(cornerRadius: 20)
                    .fill(phase == 0 ? .red : phase == 1 ? .green : .blue)
                    .frame(width: 150, height: 150)
                    .scaleEffect(phase == 1 ? 1.2 : 1.0)
            } animation: { phase in
                switch phase {
                case 0: .easeInOut(duration: 1.0)
                case 1: .spring(response: 0.6, dampingFraction: 0.7)
                default: .easeInOut(duration: 0.8)
                }
            }
            
            Text("Cycling through phases")
                .font(.headline)
        }
        .padding()
    }
}
Phase

Explanation:

  • PhaseAnimator: Cycles through predefined phases
  • Phase array: Defines the sequence of states
  • Different animations: Each phase can have its own animation
  • Complex sequences: Create sophisticated multi-step animations

Example 1: Loading Animation

A practical loading spinner with pulsating effect.

LoadingAnimation.swift
import SwiftUI

struct LoadingAnimation: View {
    @State private var isLoading = false
    @State private var progress: Double = 0
    
    var body: some View {
        VStack(spacing: 40) {
            // Loading spinner
            ZStack {
                Circle()
                    .stroke(Color.gray.opacity(0.3), lineWidth: 8)
                    .frame(width: 80, height: 80)
                
                Circle()
                    .trim(from: 0, to: 0.7)
                    .stroke(
                        LinearGradient(
                            gradient: Gradient(colors: [.blue, .purple]),
                            startPoint: .topLeading,
                            endPoint: .bottomTrailing
                        ),
                        style: StrokeStyle(lineWidth: 8, lineCap: .round)
                    )
                    .frame(width: 80, height: 80)
                    .rotationEffect(Angle(degrees: isLoading ? 360 : 0))
                    .animation(
                        .linear(duration: 1).repeatForever(autoreverses: false),
                        value: isLoading
                    )
            }
            
            // Progress bar
            VStack {
                Text("Downloading... \(Int(progress * 100))%")
                    .font(.headline)
                
                ZStack(alignment: .leading) {
                    Rectangle()
                        .frame(height: 8)
                        .foregroundColor(Color.gray.opacity(0.3))
                        .cornerRadius(4)
                    
                    Rectangle()
                        .frame(width: CGFloat(progress) * 300, height: 8)
                        .foregroundColor(.blue)
                        .cornerRadius(4)
                        .animation(.easeInOut(duration: 0.5), value: progress)
                }
                .frame(width: 300)
            }
            
            Button(isLoading ? "Stop Loading" : "Start Loading") {
                isLoading.toggle()
                if isLoading {
                    simulateProgress()
                } else {
                    progress = 0
                }
            }
            .buttonStyle(.borderedProminent)
        }
        .padding()
    }
    
    func simulateProgress() {
        guard isLoading else { return }
        
        if progress < 1.0 {
            DispatchQueue.main.asyncAfter(deadline: .now() + 0.5) {
                progress += 0.1
                simulateProgress()
            }
        } else {
            isLoading = false
        }
    }
}

Example 2: Interactive Card Flip

A 3D card flip animation with user interaction.

CardFlip.swift
import SwiftUI

struct CardFlip: View {
    @State private var flipped = false
    @State private var flipAngle: Double = 0
    
    var body: some View {
        VStack(spacing: 30) {
            // 3D Card Flip
            ZStack {
                // Front of card
                if !flipped {
                    RoundedRectangle(cornerRadius: 20)
                        .fill(LinearGradient(
                            gradient: Gradient(colors: [.blue, .purple]),
                            startPoint: .topLeading,
                            endPoint: .bottomTrailing
                        ))
                        .frame(width: 300, height: 200)
                        .overlay(
                            VStack {
                                Text("Front Side")
                                    .font(.title)
                                    .fontWeight(.bold)
                                    .foregroundColor(.white)
                                Text("Tap to flip")
                                    .foregroundColor(.white.opacity(0.8))
                            }
                        )
                        .rotation3DEffect(
                            .degrees(flipAngle),
                            axis: (x: 0, y: 1, z: 0)
                        )
                }
                
                // Back of card
                if flipped {
                    RoundedRectangle(cornerRadius: 20)
                        .fill(LinearGradient(
                            gradient: Gradient(colors: [.orange, .red]),
                            startPoint: .topLeading,
                            endPoint: .bottomTrailing
                        ))
                        .frame(width: 300, height: 200)
                        .overlay(
                            VStack {
                                Text("Back Side")
                                    .font(.title)
                                    .fontWeight(.bold)
                                    .foregroundColor(.white)
                                Text("Secret content here!")
                                    .foregroundColor(.white.opacity(0.8))
                            }
                        )
                        .rotation3DEffect(
                            .degrees(flipAngle + 180),
                            axis: (x: 0, y: 1, z: 0)
                        )
                }
            }
            .onTapGesture {
                withAnimation(.spring(response: 0.6, dampingFraction: 0.8)) {
                    flipped.toggle()
                    flipAngle += 180
                }
            }
            
            Text("Tap the card to flip it!")
                .font(.headline)
        }
        .padding()
    }
}

struct BouncingBall: View {

Exercise 2: Animated Login Form

Task: Create a login form with the following animations:

  • Form fields slide in from the right when appearing
  • Login button scales up when the form is valid
  • Error message shakes if login fails
  • Success checkmark appears with scale animation
AnimatedLoginForm.swift
import SwiftUI

struct AnimatedLoginForm: View {
    @State private var username = ""
    @State private var password = ""
    @State private var showForm = false
    @State private var isLoggingIn = false
    @State private var loginSuccess = false
    @State private var showError = false
    
    var formIsValid: Bool {
        !username.isEmpty && !password.isEmpty
    }
    
    var body: some View {
        VStack(spacing: 30) {
            if showForm {
                VStack(spacing: 20) {
                    // Username field
                    TextField("Username", text: $username)
                        .textFieldStyle(.roundedBorder)
                        .padding(.horizontal)
                        .transition(.move(edge: .trailing))
                    
                    // Password field
                    SecureField("Password", text: $password)
                        .textFieldStyle(.roundedBorder)
                        .padding(.horizontal)
                        .transition(.move(edge: .trailing))
                    
                    // Login button
                    Button(action: {
                        attemptLogin()
                    }) {
                        if isLoggingIn {
                            ProgressView()
                                .progressViewStyle(CircularProgressViewStyle(tint: .white))
                        } else if loginSuccess {
                            Image(systemName: "checkmark")
                                .font(.title2)
                        } else {
                            Text("Login")
                                .fontWeight(.semibold)
                        }
                    }
                    .frame(maxWidth: .infinity)
                    .padding()
                    .background(formIsValid ? Color.blue : Color.gray)
                    .foregroundColor(.white)
                    .cornerRadius(10)
                    .padding(.horizontal)
                    .scaleEffect(formIsValid ? 1.05 : 1.0)
                    .animation(.spring(response: 0.3, dampingFraction: 0.7), value: formIsValid)
                    .disabled(!formIsValid || isLoggingIn)
                    
                    // Error message
                    if showError {
                        Text("Invalid credentials. Please try again.")
                            .foregroundColor(.red)
                            .transition(
                                .asymmetric(
                                    insertion: .scale.combined(with: .opacity),
                                    removal: .opacity
                                )
                            )
                    }
                }
            }
            
            if !showForm {
                Button("Show Login Form") {
                    withAnimation(.spring()) {
                        showForm = true
                    }
                }
                .buttonStyle(.borderedProminent)
            }
        }
        .padding()
    }
    
    func attemptLogin() {
        isLoggingIn = true
        
        // Simulate network request
        DispatchQueue.main.asyncAfter(deadline: .now() + 2) {
            isLoggingIn = false
            
            // Simulate random success/failure
            if Bool.random() {
                loginSuccess = true
                showError = false
                
                // Reset after success
                DispatchQueue.main.asyncAfter(deadline: .now() + 1.5) {
                    withAnimation {
                        showForm = false
                        username = ""
                        password = ""
                        loginSuccess = false
                    }
                }
            } else {
                showError = true
                
                // Auto-hide error
                DispatchQueue.main.asyncAfter(deadline: .now() + 2) {
                    withAnimation {
                        showError = false
                    }
                }
            }
        }
    }
}
